Basic information 
1 Font name: Effra Pro
2 TrueType Version record: 1.001
3 Regional package: Latin
4 Supported scripts: Latin
5 Number of glyphs: 486
6 Weights/Styles: Regular, Regular Italic, Bold, Bold Italic

Technology 
1 File formats: The files have been delivered in the TrueType-based OpenType format, for use in all major Operating Systems.
2 Embedding settings: Print and preview of the document is allowed (print and preview mode); subsetting is allowed.
3 Default numerals: Tabular. Proportional figures accessible via OpenType features.
